# Portfolio Intro Page

간단한 자기소개용 정적 웹페이지입니다.  
`HTML`, `CSS`, `JavaScript`만으로 구성되어 있어 별도 설치 없이 바로 실행할 수 있습니다.

## Preview

이 페이지는 다음 내용을 포함합니다.

- 메인 자기소개 섹션
- 핵심 역량 소개
- 성장 타임라인
- 프로젝트 카드
- 연락처 영역
- 반응형 레이아웃
- 스크롤 등장 애니메이션

## Files

- `index.html`: 페이지 구조와 콘텐츠
- `style.css`: 전체 스타일, 레이아웃, 반응형 디자인
- `script.js`: 스크롤 등장 애니메이션 처리

## Run

별도 패키지 설치 없이 실행할 수 있습니다.

1. 프로젝트 폴더에서 `index.html` 파일을 브라우저로 엽니다.
2. 또는 VS Code Live Server 같은 정적 서버로 실행합니다.

## Customize

다음 항목을 본인 정보에 맞게 수정하면 바로 포트폴리오로 사용할 수 있습니다.

- 이름: `홍길동`
- 직무명
- 소개 문구
- 핵심 역량 태그
- 연혁 및 프로젝트 내용
- 이메일 주소
- GitHub / Notion / 블로그 링크

주요 수정 위치:

- 소개 문구 및 콘텐츠: `index.html`
- 색상, 폰트, 간격, 반응형 스타일: `style.css`
- 애니메이션 동작: `script.js`

## Design Notes

이 페이지는 아래 방향으로 구성했습니다.

- 강한 첫인상을 주는 타이포그래피
- 부드러운 배경 그라데이션과 카드형 레이아웃
- 모바일과 데스크톱 모두 대응 가능한 반응형 구조
- 과하지 않은 인터랙션으로 집중도 유지
